Parameters:
  • name (str) –

    Name of the molecule.

  • atoms (List[Atom]) –

    List of atoms in the molecule.

  • bonds (Dict[Atom, List[Bond]], default: None ) –

    Dictionary mapping atoms to their bonds. If not provided, bonds are created based on the covalent radii of atoms and their coordinates.
