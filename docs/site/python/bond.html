psimol.Bond

Class to represent a bond from a given atom to another.

__init__(first_atom, second_atom, order=1, aromatic=False, metallic=False)

Initialize the bond.

Parameters:
  • first_atom (Atom) –

    First atom of the bond

  • second_atom (Atom) –

    Second atom of the bond

  • order (Literal[1, 2, 3], default: 1 ) –

    Bond order (single, double or triple). Defaults to 1.

  • aromatic (bool, default: False ) –

    Whether a bond is aromatic. Defaults to False.

  • metallic (bool, default: False ) –

    Whether a bond is metallic. Defaults to False.

bond_length: float

Calculate the bond length as the Euclidean distance between atoms.

Returns:
  • float( float ) –

    Bond length.
